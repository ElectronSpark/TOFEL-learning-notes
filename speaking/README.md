## 评分标准 （加分制）：
### **Delivery**:
* **Fluency** 流利度
* **Pronunciation** 发音（区别于口音）
* **Pace** 自然语速
* **Natural inotation patterns** 断句
  
### **Language Use**:
* **Grammar** 语法
* **Vocabulary** 词汇
**不需要特别复杂或者高端的词语和表达方式**

### **Topic Development**:
* 逻辑
* 相关性
* 完整回答问题
