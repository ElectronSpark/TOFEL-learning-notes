# **综合口语**（强化）
---
## 评分标准
* **要符合题目相关要求**
* 顺序（不能插叙）
* 逻辑 Campus

## 阅读步骤
* 听提示，确定主题
* 看标题，确定主题（可以看出百分之八十的原因）
  * 一定会有两个原因，或一个原因和一个条件
* 边看变做笔记
  * 如果有个点没听到，在听力材料里推断
  * 重点在于语言的流利度，大意没问题即可

## 结构
* 形式（先确定是个人还是学校）
  * 个人建议
  * 校方公告或者计划
* 结构
  * 两个原因
  * 一个原因和一个条件
  * 两个建议
* 建议、公告定位
* 原因定位

>十五秒钟内说完:<br>
>    the university(student) anoucnes(suggests),<br>
>    because reason 1,<br>
>    plus reason 2,<br>
> 遇到说明特定时间不用特别注意，直接用"the certain times"

## 听力结构
* conversation（男/女）
* 态度（要么赞同要么反对） + 原因1 + 原因2
  * ...
* 转述句型
  * the man/woman in this conversation - thinks it's a good/bad idea
  * If this plan goes through
  * 说的时候用**because ... plus ...**
  * A means ...
